# Fraluc06 Tap

## How do I install these formulae?

`brew install fraluc06/tap/<formula>`

Or `brew tap fraluc06/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "fraluc06/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
